# OpenFantasy의 혼종 생명체들

> *"서로 다른 본성의 혼합 속에서 그 부분들의 합보다 더 위대한 무언가가 나타난다. 그것은 대조적인 노래들이 하나의 멜로디로 연주되는 조화이다."*

## 개요

OpenFantasy에서의 혼종 생명체는 여러 종이나 생명체의 범주의 특징을 결합한 생물을 나타냅니다. 이러한 독특한 존재는 서로 다른 기원에서 비롯된 생리적, 마법적, 때로는 영적인 특성을 가지고 있으며, 종종 특수한 생태학적 또는 사회적 틈새를 차지하는 독특한 존재를 만들어냅니다.

밀접하게 관련된 종 사이의 단순한 이종 교배와는 달리, OpenFantasy의 진정한 혼종은 일반적으로 수생과 육생, 인간형과 동물형, 심지어 물질과 원소와 같은 근본적으로 다른 존재 범주를 연결합니다. 이러한 결합은 부모 범주에서 발견되지 않는 독특한 원소 구성과 특수 능력을 초래합니다.
