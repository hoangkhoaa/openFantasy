# OpenFantasy (オープンファンタジー)の混成生物

> *"異なる性質の融合は、その構成要素の総和を超える何かを生み出す。それは、対照的な歌が1つの旋律として奏でられる調和である。"*

## 概要

OpenFantasy (オープンファンタジー)における混成生物とは、複数の種や生命のカテゴリーの特徴を組み合わせた生物のことです。これらのユニークな存在は、生理学的、魔法的、そして時には精神的な特性を異なる起源から持ち合わせており、独特な存在として、しばしば特殊な生態学的または社会的なニッチを占めています。

近縁種間の単純な交配種とは異なり、OpenFantasy (オープンファンタジー)における真の混成種は、通常、水生と陸生、人型と動物、あるいは物質と元素など、根本的に異なる存在のカテゴリーを橋渡しします。これらの組み合わせは、親カテゴリーのいずれにも見られない、ユニークな元素構成と特別な能力をもたらします。
