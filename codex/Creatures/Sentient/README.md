## 概述 (Overview)

OpenFantasy中的有知觉物种 (Sentient species in OpenFantasy)是具有高等智慧、自我意识和形成复杂社会能力的生物。 他们拥有通过复杂的语言进行交流、创造文化、发展技术以及通过有意识的努力（而不仅仅是本能）来操纵魔法的能力。

有知觉物种 (Sentient species)与其他生物的区别在于其较高的灵魂成分 (high soul component)，这使得他们能够进行抽象思考、道德推理和形成文明。 每个有知觉物种 (Sentient species)都根据其元素构成和所居住的环境进化出独特的适应性和社会结构。

所有有知觉物种 (Sentient species)都具有某些基本特征：

- **社会组织 (Social Organization)**：形成具有不同角色和等级制度的复杂社会
