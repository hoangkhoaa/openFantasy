# Welcome to the OpenFantasy System Reference Document!

This document details the rules and mechanics of the OpenFantasy role-playing game system. OpenFantasy is designed to be a flexible and adaptable system, suitable for a wide range of fantasy settings and play styles.

## Core Concepts

*   **Attributes (属性)**: Every character has six core Attributes (属性): Strength (力量), Dexterity (敏捷), Constitution (体质), Intelligence (智力), Wisdom (智慧), and Charisma (魅力). These Attributes (属性) define a character's innate capabilities.

*   **Skills (技能)**: Skills (技能) represent a character's learned abilities. Skills (技能) are tied to specific Attributes (属性) and are used to resolve actions.

*   **Checks (检定)**: Checks (检定) are used to determine the outcome of an action. A check (检定) involves rolling a 20-sided die (d20), adding relevant modifiers, and comparing the result to a target number (TN).

*   **Target Number (TN) (目标数值)**: The Target Number (TN) (目标数值) represents the difficulty of a task. To succeed at a check (检定), the result must meet or exceed the Target Number (TN) (目标数值).

## Character Creation

1.  **Determine Attributes (属性) (确定属性)**: Roll 4d6, drop the lowest die, and sum the remaining three dice for each Attribute (属性). Assign the results to your desired Attributes (属性).

2.  **Choose a Race (种族)**: Select a Race (种族) for your character. Each Race (种族) provides specific benefits and abilities.

3.  **Choose a Class (职业)**: Select a Class (职业) for your character. Each Class (职业) provides unique skills and abilities.

4.  **Determine Skills (技能) (确定技能)**: Allocate skill points to your desired Skills (技能).

5.  **Equip your Character (装备角色)**: Choose starting equipment for your character.

*   **Skills (技能)**: Skills (技能) 代表一个角色所学到的能力。Skills (技能) 与特定的 Attributes (属性) 相关联，并用于解决行动。

*   **Checks (检定)**: Checks (检定) 用于确定行动的结果。 一个 check (检定) 涉及掷一个 20 面骰子 (d20)，加上相关的修正值，并将结果与目标数值 (TN) 进行比较。

*   **Target Number (TN) (目标数值)**: Target Number (TN) (目标数值) 代表任务的难度。 要成功通过 check (检定)，结果必须达到或超过 Target Number (TN) (目标数值)。

5.  **Equip your Character (装备角色)**: 为你的角色选择起始装备。
