# OpenFantasyのモンスター (Monsters of OpenFantasy)

> *"月影の下を歩むもの全てが、自然の摂理によって生まれたわけではない。闇、堕落、あるいは現実そのものの熱に浮かされた夢から現れる生物もいるのだ。"*

## 概要 (Overview)

OpenFantasyにおいて、「モンスター (monsters)」とは、通常の生態系のバランスから外れた、危険または不自然な生物を指します。自然のプロセスを通じて進化した動物とは異なり、モンスター (monsters)は通常、魔法的な事故、堕落、意図的な創造、または他の次元からの出現によって生じます。

モンスター (monsters)は、従来の理解を超えた能力を持っていることが多く、それらが知的文明や自然生態系にとって特に危険な存在となっています。普遍的に悪意があるわけではありませんが、その捕食的な性質、縄張りへの攻撃性、または自然界との非互換性により、ほとんどのモンスター (monsters)は重大な脅威となります。
