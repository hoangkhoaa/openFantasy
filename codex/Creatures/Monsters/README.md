# OpenFantasy的怪物 (Monsters of OpenFantasy)

> *"并非所有在月光下行走之物皆由自然设计而生。有些生物从黑暗、腐败，或现实本身狂热的梦境中涌现。"*

## 概述 (Overview)

在 OpenFantasy 中，“怪物 (monsters)”指的是存在于正常生态平衡之外的危险或不自然的生物。与通过自然过程进化的动物不同，怪物 (monsters) 通常通过魔法事故、腐败、蓄意创造或从其他存在层面显现而产生。

怪物 (monsters) 通常拥有违背传统理解的能力，这使得它们对有知觉的文明和自然生态系统都特别危险。虽然并非普遍怀有恶意，但由于它们的掠食性、领土侵略性或与自然世界的不相容性，大多数怪物 (monsters) 构成了重大威胁。

## 怪物起源 (Monster Origins)

怪物 (monsters) 通过多种方式产生，每种方式都创造出不同类型的生物：

| Origin (起源) | Description (描述) | Examples (例子) |
|--------|-------------|----------|
| **Corruption (腐败)** | 被黑暗魔法或污染扭曲的自然生物 | 受损的狼 (Blighted wolves)、被污染的元素 (tainted elementals)、瘟疫兽 (plague beasts) |
| **Magical Creation (魔法创造)** | 通过法术刻意制作 | 奇美拉 (Chimeras)、人造憎恶 (artificial abominations)、束缚的灵魂 (bound spirits) |
| **Planar Intrusion (位面入侵)** | 从其他存在层面进入的实体 | 虚空潜行者 (Void stalkers)、梦魇 (dream haunts)、维度异常 (dimensional anomalies) |
| **Ancient Evolution (远古进化)** | 当前自然秩序之前的原始生物 | 深渊恐惧 (Deep terrors)、史前掠食者 (prehistoric predators)、活化石 (living fossils) |
| **Death Perversion (死亡扭曲)** | 通过干预自然死亡形成的生物 | 不死生物 (Undead)、灵魂束缚的躯壳 (soul-bound husks)、亡灵构装体 (necromantic constructs) |
