# Monsters of OpenFantasy (OpenFantasy의 몬스터)

> *"달 아래를 걷는 모든 것이 자연의 설계로 태어난 것은 아니다. 어떤 생물은 어둠, 타락, 또는 현실 자체의 열병 같은 꿈에서 나타난다."*

## Overview (개요)

OpenFantasy (OpenFantasy)에서 "몬스터(monsters)"는 정상적인 생태계 균형 밖에 존재하는 위험하거나 부자연스러운 생물을 지칭합니다. 자연적인 과정을 통해 진화한 동물과는 달리, 몬스터(monsters)는 일반적으로 마법적인 사고, 타락, 의도적인 창조, 또는 다른 차원에서의 발현을 통해 나타납니다.

몬스터(monsters)는 종종 기존의 이해를 벗어나는 능력을 가지고 있어 지적 문명과 자연 생태계 모두에게 특히 위험합니다. 보편적으로 악한 것은 아니지만, 대부분은 포식적인 본성, 영역 침략, 또는 자연 세계와의 양립 불가능성으로 인해 상당한 위협을 제기합니다.

| **Magical Creation (마법적 창조)** | 주문 제작을 통해 의도적으로 만들어짐 | Chimeras (키메라), artificial abominations (인공 흉물), bound spirits (속박된 영혼) |
