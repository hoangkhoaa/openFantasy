# OpenFantasy的魔法生物 (Magical Beings of OpenFantasy)

> *"他们是被赋予形态和目的的魔法，是我们世界中流动的神秘力量的活生生的化身。"*

## 概述 (Overview)

OpenFantasy中的魔法生物 (Magical beings in OpenFantasy) 是从根本上由魔法能量构成、由魔法能量创造或注入魔法能量的生物。与自然进化并且仅仅使用魔法的智慧物种不同，魔法生物 (magical beings) 将魔法作为其存在的内在组成部分，通常它们的物理形态主要用作魔法能量的容器。

这些生物 (beings) 的特点是它们异常高的 Mana Capacity (魔法容量)，通常超过自然生物 (natural creatures)。许多生物 (beings) 起源于强烈的魔法活动时期，或者通过有意的魔法过程创造，而另一些则代表了特定魔法概念或元素的物理表现。
