## Core Principles (核となる原則)

Force Vector (力のベクトル) は、純粋なマナを方向性のある運動エネルギーに変換する基本的なユーティリティスペルです。目に見える効果を示すエレメンタルスペルとは異なり、Force Vector (力のベクトル) は目に見えない力の線を作成し、キャスターの意図と使用する特定のパターンバリエーションに基づいて、物体を押し、引き、持ち上げ、または操作できます。

この呪文は、キャスターのマナと物理的な物質の間に共鳴接続を作成することによって機能し、直接的な物理的接触なしに運動エネルギーの伝達を可能にします。コアパターンは以下を確立します。

1. **Force Origin Point (力の発生点)**: The anchor from which force emanates (力が湧き出る起点)
2. **Vector Direction (ベクトルの方向)**: The precise angle and orientation of the force (力の正確な角度と方向)
3. **Intensity Matrix (強度マトリックス)**: The strength and distribution of force along the vector (ベクトルに沿った力の強さと分布)
4. **Decay Function (減衰関数)**: How quickly the force dissipates with distance (距離とともに力がどれだけ速く減衰するか)

Force Vector (力のベクトル) が他の多くの呪文と異なるのは、高いマナの purity (純度) よりも適切な technique (技術) に依存していることです。 moderate (中程度) の purity level (純度レベル) の practitioners (実践者) であっても、規律ある training (訓練) と正確な execution (実行) によって目覚ましい結果を達成できます。そのため、マナの purification (浄化) を専門としていない戦士や実用的な魔術師に特に人気があります。

## Standard Variants (標準バリアント)

### Force Push (フォースプッシュ)
The most basic application (最も基本的な応用), creating a directed push away from the caster (キャスターから離れる方向への押し出しを作成する):
- **Difficulty (難易度)**: Novice (初心者)
- **Purity Requirement (純度要件)**: Basic (基本) (15-20%)
- **Effect (効果)**: Generates a short burst of force in a specified direction (指定された方向に短い力のバーストを生成する)
- **Typical Users (一般的な使用者)**: Warriors (戦士), Guards (警備兵), Duelists (決闘者)
- **Applications (応用)**: Repelling attackers (攻撃者の撃退), creating distance (距離の作成), knocking back objects (オブジェクトのノックバック)

### Force Pull (フォースプル)
Reverses the vector (ベクトルを反転させる) to draw objects toward the caster (キャスターに向かってオブジェクトを引き寄せる):
- **Difficulty (難易度)**: Novice (初心者)
- **Purity Requirement (純度要件)**: Basic (基本) (15-20%)
- **Effect (効果)**: Creates attractive force to draw small objects to hand (小さな物体を手元に引き寄せる引力を生成する)
- **Typical Users (一般的な使用者)**: Rangers (レンジャー), Duelists (決闘者), Scholars (学者)
- **Applications (応用)**: Retrieving weapons (武器の回収), gathering items (アイテムの収集), disarming opponents (対戦相手の武装解除)

### Gravity Reduction (重力軽減)
Manipulates the force vector (力のベクトルを操作する) counter to natural gravity (自然重力に対抗する):
- **Difficulty (難易度)**: Intermediate (中級)
- **Purity Requirement (純度要件)**: Basic to Refined (基本から洗練) (20-35%)
- **Effect (効果)**: Reduces effective weight of target by 30-70% (ターゲットの有効重量を30〜70％削減する)
- **Typical Users (一般的な使用者)**: All classes (すべてのクラス) (specialized applications (特殊な応用))
- **Applications (応用)**: Lightening carried equipment (携帯機器の軽量化), enhancing jumps (ジャンプの強化), enabling stealth movement (ステルス移動の有効化)
