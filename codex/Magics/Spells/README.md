## Categorías de Hechizos

| Category (Categoría) | Description (Descripción) |
|----------|-------------|
| **Elemental** | Manipular elementos primarios (Fire (Fuego), Water (Agua), Earth (Tierra), Air (Aire), Light (Luz), Shadow (Sombra), Life (Vida), Death (Muerte)) |

| **Transformation** (Transformación) | Cambiar la forma física o las propiedades de los objetivos |
| **Summoning** (Invocación) | Llamar criaturas, entidades u objetos desde otro lugar |

Minimum purity level (Nivel de pureza mínimo) needed to cast successfully (necesario para lanzar con éxito)

#### C. Pattern Structure (Estructura del Patrón)

### 3. Información Práctica

- Common applications (Aplicaciones comunes)

- **Above minimum** (Por encima del mínimo): Gains enhanced effectiveness (Gana mayor efectividad), precision (precisión), or duration (o duración)

## Character Progression (Progresión del Personaje)

### Spell Mastery Path (Ruta de Maestría de Hechizos)

1. **Novice** (Novato): Basic purity (Pureza básica) (10-35%), simple patterns (patrones simples)
2. **Adept** (Adepto): Refined purity (Pureza refinada) (35-65%), moderate complexity (complejidad moderada)
3. **Expert** (Experto): Superior purity (Pureza superior) (65-85%), complex patterns (patrones complejos)
4. **Master** (Maestro): Perfect purity (Pureza perfecta) (85%+), intricate patterns (patrones intrincados)

### Key Development Areas (Áreas Clave de Desarrollo)

- **Capacity** (Capacidad): Increasing mana reserves (Aumentar las reservas de maná)
- **Purification** (Purificación): Improving mana quality (Mejorar la calidad del maná)
- **Pattern Knowledge** (Conocimiento de Patrones): Learning more complex spells (Aprender hechizos más complejos)
- **Specialization** (Especialización): Focusing on particular mana types (Centrarse en tipos de maná particulares)
