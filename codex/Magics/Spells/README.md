## 주문 범주 (Spell Categories)

| Category (범주) | Description (설명) |
|----------|-------------|
| **Elemental (원소)** | Manipulate primary elements (Fire (불), Water (물), Earth (흙), Air (공기), Light (빛), Shadow (그림자), Life (생명), Death (죽음)) |
| **Utility (유틸리티)** | Practical spells for everyday tasks and problem-solving (일상 업무 및 문제 해결을 위한 실용적인 주문) |
| **Protection (보호)** | Defensive magic that shields or counters hostile forces (적대적인 힘을 막거나 방어하는 마법) |
| **Divination (점술)** | Reveal information, enhance perception, provide insight (정보를 드러내고, 인식을 향상시키며, 통찰력을 제공) |
| **Illusion (환상)** | Create false sensory impressions or alter perception (거짓 감각 인상을 만들거나 인식을 변경) |
| **Enchantment (마법 부여)** | Imbue objects or beings with magical properties (물체나 존재에 마법 속성을 부여) |
| **Transformation (변환)** | Change physical form or properties of targets (대상의 물리적 형태 또는 속성을 변경) |
| **Summoning (소환)** | Call creatures, entities, or objects from elsewhere (다른 곳에서 생물, 존재 또는 물체를 소환) |

## 문서 형식 (Documentation Format)

각 주문 문서는 다음 구조를 따릅니다.

### 1. 기본 정보 (Basic Information)
- Spell name (주문 이름) and overview (개요)
- Casting information (시전 정보) (difficulty (난이도), time (시간), duration (지속 시간), range (범위), components (구성 요소))
- Technical details (기술적 세부 사항) (mana types (마나 유형), purity requirements (순도 요구 사항), complexity (복잡성))

### 2. 기술적 세부 사항 (Technical Details)

#### A. 마나 유형 분포 (Mana Type Distribution)
Proportions of mana types (마나 유형의 비율) (e.g., "Base Mana (기본 마나) 60%, Umbramana (움브라마나) 30%, Luxmana (룩스마나) 10%")

#### B. 순도 요구 사항 (Purity Requirements)
Minimum purity level needed to cast successfully (성공적으로 시전하는 데 필요한 최소 순도 수준)

#### C. 패턴 구조 (Pattern Structure)
Magical configuration that creates the spell effect (주문 효과를 생성하는 마법 구성)

### 3. 실용적인 정보 (Practical Information)
- Effects (효과) and limitations (제한 사항)
- Common applications (일반적인 적용)
- Variations (변형) and adaptations (적응)

### 4. 학습 및 위험 (Learning and Risks)
- Risks (위험) and miscast effects (잘못된 시전 효과)
- Learning progression (학습 과정)
- Historical context (역사적 맥락)

### 주문 시전 효과 (Effects on Spellcasting)

- **Below minimum (최저 미만)**: Spell fails or produces unpredictable results (주문이 실패하거나 예측할 수 없는 결과가 발생)
- **At minimum (최저 수준)**: Functions but may be less effective or stable (작동하지만 효과적이거나 안정적이지 않을 수 있음)
- **Above minimum (최저 초과)**: Gains enhanced effectiveness, precision, or duration (향상된 효과, 정확성 또는 지속 시간을 얻음)

2. **Adept (숙련자)**: Refined purity (정제된 순도) (35-65%), moderate complexity (적당한 복잡성)
