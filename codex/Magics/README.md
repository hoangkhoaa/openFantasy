| 원소 | 반대 원소 | 관련 마나 |

## 강화 및 발전

마법 능력에서 캐릭터의 발전은 다음과 같은 여러 경로를 따릅니다.

1. **용량 개발**: 캐릭터가 보유할 수 있는 마나의 양 증가
2. **정화 기술**: 캐릭터가 생산할 수 있는 마나의 품질 향상
3. **패턴 복잡성**: 더 복잡한 주문 구조 학습
4. **원소 조율**: 특정 원소와의 친화력 개발
5. **전문화**: 특정 마법 분야에 집중

강화 버프는 이러한 능력을 자연적 한계를 넘어 일시적으로 향상시킬 수 있습니다.
