## Spectrum Overview (Visión General del Espectro)

The Elemental Mana Spectrum (El Espectro de Maná Elemental) representa la gama completa de tipos de maná especializados que pueden derivarse del [Base Mana](/codex/Magics/Core/BaseMana.md) (Maná Base) neutral a través del proceso de [Mana Specialization](/codex/Magics/Core/ManaSpecialization.md) (Especialización de Maná). Este espectro sigue patrones y relaciones específicas que determinan cómo las energías elementales interactúan, se combinan y se oponen entre sí. Comprender estas relaciones es esencial para el lanzamiento de hechizos avanzado, particularmente cuando se trabaja con múltiples elementos simultáneamente.

### Visualization of the Spectrum (Visualización del Espectro)

The Elemental Spectrum (El Espectro Elemental) puede visualizarse de varias maneras:

1. **The Chromatic Circle** (El Círculo Cromático): Una disposición circular que muestra las transiciones y oposiciones elementales
2. **The Harmonic Scale** (La Escala Armónica): Una representación lineal que enfatiza las relaciones de resonancia

4. **The Wavelength Continuum** (El Continuo de Longitud de Onda): Una representación de los elementos como frecuencias variables de Base Mana (Maná Base)

Cada visualización ofrece diferentes perspectivas sobre las relaciones elementales.

### The Specialization Gradient (El Gradiente de Especialización)

La transición de Base Mana (Maná Base) a formas elementales totalmente especializadas no es binaria, sino que se produce a lo largo de un gradiente:

| Specialization Level (Nivel de Especialización) | Base Mana Content (Contenido de Maná Base) | Elemental Character (Carácter Elemental) | Typical Application (Aplicación Típica) |
|----------------------|-------------------|---------------------|---------------------|

| 1-25% | 75-99% | Subtle influence (Influencia sutil) | Gentle effects, broad-spectrum magic (Efectos suaves, magia de amplio espectro) |
| 25-50% | 50-75% | Clear influence (Influencia clara) | Standard magical workings (Operaciones mágicas estándar) |
| 50-75% | 25-50% | Dominant character (Carácter dominante) | Element-focused spellcasting (Lanzamiento de hechizos centrado en elementos) |

| 90-100% | 0-10% | Pure elemental form (Forma elemental pura) | Elemental manifestations, summonings (Manifestaciones elementales, invocaciones) |

Este gradiente explica por qué muchos hechizos requieren proporciones específicas de Base Mana (Maná Base) y tipos elementales en lugar de formas puras.
