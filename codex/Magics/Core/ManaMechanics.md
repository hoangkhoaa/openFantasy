## 마나 용량과 주문 시전

존재의 마나 용량은 마법 에너지를 보유하고 흐르게 하는 타고난 능력을 나타냅니다. 이 요소 구성 요소는 주문 시전 능력에 여러 가지 중요한 영향을 미칩니다.

### 저장 용량

| 마나 용량 | 최대 마나 부피 | 실제 효과 |
|---------------|---------------------|------------------|
| 5-15% | 매우 낮음 | 가장 간단한 주문만 시전할 수 있으며 빠르게 소모됨 |
| 15-30% | 낮음 | 기본적인 주문과 제한적인 시전 세션을 처리할 수 있음 |
| 30-50% | 중간 | 중간 수준의 주문과 적당한 지속 시간을 관리할 수 있음 |
| 50-70% | 높음 | 고급 주문을 시전하고 여러 효과를 유지할 수 있음 |
| 70-85% | 매우 높음 | 복잡한 마법 작업과 장시간 세션을 처리할 수 있음 |
| 85%+ | 뛰어남 | 전설적인 주문을 시전하고 영구적인 효과를 유지할 수 있음 |

## 마나 순도와 주문 효과

날것의 [야생 마나](/codex/Magics/WildMana.md)는 주문 시전에 효과적으로 사용하려면 정제해야 합니다. 광석을 금속으로 정제하는 것처럼 이 과정은 혼돈스러운 야생 마나를 패턴 형성에 적합한 제어된 정제된 마나 유형으로 변환합니다.

### 순도 계산

순도는 다음 공식을 사용하여 계산됩니다.

```
순도 백분율 = (정제된 마나) / (정제된 마나 + 야생 마나) × 100%
```

이 수학적 관계는 야생 마나 함량이 주문 효과에 반비례하는 이유를 설명합니다.

### 순도 수준

| 순도 수준 | 백분율 | 야생 마나 함량 | 달성 난이도 |
|--------------|------------|-------------------|----------------------|
| **조잡** | 0-10% | 90-100% | 최소한의 훈련 필요 |
| **기본** | 10-35% | 65-90% | 기본 훈련 (1-2년) |
| **정제** | 35-65% | 35-65% | 중간 훈련 (3-5년) |
| **우수** | 65-85% | 15-35% | 고급 훈련 (6-10년) |
| **완벽** | 85-100% | 0-15% | 숙련 훈련 (10년 이상) |

### 마나 용량과 최대 달성 가능한 순도

시전자의 마나 용량은 잠재적으로 달성할 수 있는 최대 순도 수준에 영향을 미칩니다.

| 마나 용량 | 최대 달성 가능한 순도 | 최소 야생 마나 |
|---------------|-----------------------------|-------------------|
| 5-15% | 기본 (최대 25%) | 75%+ |
| 15-30% | 기본에서 정제 (최대 40%) | 60%+ |
| 30-50% | 정제 (최대 60%) | 40%+ |
| 50-70% | 우수 (최대 80%) | 20%+ |
| 70-85% | 완벽 (최대 95%) | 5%+ |
| 85%+ | 완벽 (최대 100%) | 미량 |

이 제한은 자연적으로 높은 마나 용량을 가진 종족 (예: 엘프 및 페이)이 높은 순도를 요구하는 복잡한 마법에 능숙한 반면, 낮은 용량을 가진 종족 (예: 드워프)은 일반적으로 낮은 순도를 요구하지만 안정성을 중시하는 마법에 집중하는 이유를 설명합니다.

4. **특수화**: 기본 마나를 원소 유형으로 추가 변환하기

종과 개인은 각 단계에서 효율성이 다르므로 마나 용량이 비슷한 경우에도 마법 적성의 변화를 설명합니다.

- **원소 마나**: 특정 마법 효과를 제공하는 전문 에너지 (파이로마나, 하이드로마나 등)

기본 마나와 전문 마나 유형의 평균 비율은 주문 난이도와 관련이 있습니다.

- **심각한 부족**: 주문 패턴이 불안정해져 잠재적으로 해로운 부작용을 일으킴
- **치명적인 부족**: 패턴이 파괴적으로 붕괴되어 잠재적으로 시전자에게 해를 끼침

## 원소의 조화

시전자의 마나 용량, 달성된 순도 수준 및 주문의 요구 사항 간의 관계는 음악적 비유를 통해 이해할 수 있습니다.

- **마나 용량**은 음악가가 연주할 수 있는 음역과 같습니다.
- **순도 수준**은 각 음을 얼마나 정확하게 낼 수 있는지와 같습니다.
- **야생 마나 함량**은 음악을 방해하는 배경 소음과 같습니다.
- **주문 요구 사항**은 특정 음악 작품의 요구와 같습니다.

음역이 제한된 음악가가 자신의 음역을 벗어나는 음표가 필요한 곡을 연주할 수 없는 것처럼, 마나 용량이 불충분한 마법사는 고급 주문이 요구하는 순도 수준을 달성할 수 없습니다. 마찬가지로, 용량이 큰 시전자도 야생 마나 간섭을 효과적으로 최소화하는 기술 (정화 제어)을 개발해야 합니다.
