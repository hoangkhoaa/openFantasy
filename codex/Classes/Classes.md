## 概要

OpenFantasy の各クラスは、[**Exanthis**](/codex/Basic/Exanthis.md)、[**Souls**](/codex/Basic/Soul.md)、および **Mana Capacity** のユニークな調和を表し、世界での役割を定義する能力のシンフォニーを生み出します。

## クラス構造

クラスは主なカテゴリに編成され、それぞれに、コアクラスから分岐する多数のサブクラスまたはキャリアパスがあります。これらのサブクラスは、音楽のテーマのバリエーションのように、クラスの基本的な能力の特殊な応用を表しています。

## クラスの進行

OpenFantasy のクラスは、個人が時間をかけて能力を開発できるようにする進行システムに従います。この進行はレベルで表され、各レベルはミュージシャンがますます複雑な作品を学ぶように、新しい能力と力を利用できるようになります。

### レベルティア

| ティア | レベル | 説明 |
|------|--------|-------------|
| **Novice** | 1-5 | 基本を学ぶ初心者 |
| **Adept** | 6-10 | 基本をマスターし、専門分野を開発する者 |
| **Expert** | 11-15 | 高度に磨かれた能力を持つスペシャリスト |
| **Master** | 16-20 | 真の熟練に達した者 |
| **Legendary** | 21+ | 通常の制限を超える個人 |

## マルチクラス

一部の個人は、複数のクラスを追求することを選択し、異なる道からの能力を組み合わせてユニークな組み合わせを作成します。マルチクラスとして知られるこの慣行は、より優れた汎用性を可能にしますが、ミュージシャンが複数の楽器を演奏しても、単一の楽器をマスターできない可能性があるように、単一のクラスの専門分野の深さを制限する可能性があります。

## クラスと種族の相互作用

種族が異なれば、元素組成に基づいて特定のクラスに対して自然な親和性があります。

| 種族 | 自然なクラス親和性 |

| **Humans** | すべてのクラスに対してバランスの取れた親和性 |
| **Elves** | メイジ、レンジャー、バードクラスに対して強い親和性 |
| **Dark Elves** | メイジ、ローグ、ワーロッククラスに対して強い親和性 |
| **Dwarves** | ウォーリアー、アーティフィサー、クレリッククラスに対して強い親和性 |
| **Beastfolk** | レンジャー、ドルイド、ウォーリアークラスに対して強い親和性 |
| **Dragonkin** | ウォーリアー、メイジ、ワーロッククラスに対して強い親和性 |
