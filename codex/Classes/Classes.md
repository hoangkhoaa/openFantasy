## 概述 (Gàishù)

OpenFantasy 中的每个职业都代表着 [**Exanthis**](/codex/Basic/Exanthis.md)、[**Souls**](/codex/Basic/Soul.md) 和 **Mana Capacity** 的独特和谐，创造了一种能力的交响曲，定义了它们在世界中的角色。

## 职业结构 (Zhíyè Jiégòu)

职业被组织成主要类别，每个类别都有许多子职业或职业道路，从核心职业中分支出来。这些子职业代表了该职业基本能力的专门应用，就像音乐主题的变奏一样。

## 职业进阶 (Zhíyè Jìnjiē)

OpenFantasy 中的职业遵循一种进阶系统，该系统允许个人随着时间的推移发展自己的能力。这种进阶由等级表示，每个等级都可以访问新的能力和力量，就像一位音乐家学习越来越复杂的作品一样。

### 等级分层 (Děngjí Fēncéng)

| Tier (级别) | Levels (等级) | Description (描述) |
|------|--------|-------------|
| **Novice** (新手) | 1-5 | 学习基础知识的初学者 |
| **Adept** (熟手) | 6-10 | 掌握基础知识，发展专业化 |
| **Expert** (专家) | 11-15 | 具有高度磨练能力的专家 |
| **Master** (大师) | 16-20 | 那些已经达到真正掌握的人 |
| **Legendary** (传奇) | 21+ | 超越正常限制的个人 |

## 兼职 (Jiānzhí)

有些人选择追求多个职业，将来自不同道路的能力结合起来以创造独特的组合。这种做法称为兼职，可以实现更大的多功能性，但可能会限制任何单一职业的专业化深度，就像一位演奏多种乐器的音乐家，但可能无法掌握任何一种乐器一样。
