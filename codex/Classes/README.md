## 兼职

角色可以同时追求多个职业道路，创造独特的能力组合。这种多功能性以牺牲在任何单一学科中的专业掌握为代价。

### 常见的兼职组合

| 组合 | 协同效应 |
|-------------|---------|
| 战士/法师 (Warrior/Mage) | 物理战斗与魔法能力相结合 |
| 牧师/战士 (Cleric/Warrior) | 神术与武术相结合 |
| 盗贼/游侠 (Rogue/Ranger) | 潜行与荒野专长相结合 |
| 吟游诗人/术士 (Bard/Warlock) | 表演魔法与契约能力相结合 |
| 德鲁伊/武僧 (Druid/Monk) | 自然魔法与身体训练相结合 |

## 元素构成

每个职业都具有塑造其能力的特征性元素构成：

| 职业 | Exanthis | Soul | Mana Capacity | 主要魔法亲和力 |
|-------|----------|------|---------------|--------------------------|
| **战士 (Warrior)** | 50-70% | 20-35% | 10-20% | Terramana (稳定), Base Mana (力量) |
| **法师 (Mage)** | 10-25% | 15-35% | 50-70% | 因专长而异 |
| **盗贼 (Rogue)** | 30-45% | 40-60% | 10-25% | Umbramana (隐蔽), Aeromana (机动性) |
| **牧师 (Cleric)** | 20-35% | 30-50% | 25-45% | Luxmana (光), Vitamana (治疗) |
| **游侠 (Ranger)** | 35-55% | 30-45% | 15-30% | Vitamana (自然), Aeromana (移动) |
| **工匠 (Artificer)** | 25-40% | 20-35% | 35-55% | Crystallmana (结构), Terramana (坚固) |
| **吟游诗人 (Bard)** | 15-30% | 35-55% | 25-45% | Fulgumana (声音), Aeromana (沟通) |
| **德鲁伊 (Druid)** | 20-35% | 30-45% | 30-50% | Vitamana (生长), Terramana (土地) |
| **武僧 (Monk)** | 40-55% | 30-50% | 10-25% | Base Mana (内在能量), Aeromana (移动) |
| **术士 (Warlock)** | 15-30% | 25-40% | 40-60% | Aetheramana (虚空), 因契约而异 |

这种元素平衡直接影响着一个职业对某些专业魔法形式的天然亲和力及其整体魔法能力：

- **专精效率**：Mana Capacity 较高的职业通常能达到更好的元素专精率
- **魔法净化**：Soul 与 Mana Capacity 的比率影响着一个职业将 Wild Mana 净化为 Base Mana 的能力
- **魔法耐力**：较高的 Exanthis 提供更好的身体耐力，但通常会降低魔法效率
- **混合潜力**：具有更平衡构成的职业通常擅长结合不同的元素类型
